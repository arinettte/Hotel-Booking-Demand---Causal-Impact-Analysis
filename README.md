# Hotel-Booking-Demand---Causal-Impact-Analysis
Этот проект демонстрирует применение методов причинно-следственного анализа (Causal Inference) к реальному датасету из тревел-индустрии. Используются: Difference-in-Differences (DiD) - классический эконометрический метод, Google CausalImpact (BSTS) - байесовская структурная модель временных рядов.
